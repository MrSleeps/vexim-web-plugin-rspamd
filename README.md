# VExim Plugin RSpamd Plugin
This module is designed to work with the [VExim Web UI](https://github.com/MrSleeps/VExim-Web-UI).

It is the plugin for RSpamd, currently showing Spam Stats.

You need to install [VExim Web UI](https://github.com/MrSleeps/VExim-Web-UI) for this to work

##Install notes:

You will need to run `php artisan migration` after installing
